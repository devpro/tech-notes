# MongoDB .NET Client

MongoDB provides a driver, and an Entity Framework provider, to interact with a MongoDB database from an .NET application.

📌 [Documentation](https://www.mongodb.com/docs/languages/csharp/)

## Drivers

The MongoDB .NET Driver is a NuGet package that can be used to connect to and communicate with a MongoDB cluster.

🌐 [Documentation](https://www.mongodb.com/docs/drivers/csharp/current/), [Code](https://github.com/mongodb/mongo-csharp-driver)

Release notes:

- [Inline notes](https://www.mongodb.com/docs/drivers/csharp/current/reference/release-notes/) (Since 2.25)
- [Previous links](https://www.mongodb.com/docs/drivers/csharp/current/reference/previous-versions/) (From 2.0 to 2.18)

## Entity Framework Core Provider

MongoDB Entity Framework (EF) Core Provider, in a .NET application, serves as an object-relational mapper (ORM) to work with data in a MongoDB cluster.

🌐 [Documentation](https://www.mongodb.com/docs/entity-framework/current/), [Code](https://github.com/mongodb/mongo-efcore-provider)

> [!NOTE]  
> This provider, introduced in May 2024, is now generally available.
> See the [announcement](https://www.mongodb.com/company/blog/product-release-announcements/mongodb-provider-entity-framework-core-now-generally-available).

## Analyzer

MongoDB provides a Roslyn Analyzer that checks C# code and helps understand how the code translate to MongoDB calls.

🌐 [Documentation](https://www.mongodb.com/docs/mongodb-analyzer/current/)
