# MongoDB 8.0

This minor version was released on October 2, 2024.

🌐 [Update](https://www.mongodb.com/products/updates/mongodb-8-0-is-now-available/)

## Changes

- Major performance improvements (32% higher throughput)
- Controls for handling usage spikes (improved horizontal scaling)
- Embedded sharding configuration servers
- Move collections to shards without a shard key
- Queryable Encryption: Range (expanded queryable encryption capabilities)
- Faster resharding
