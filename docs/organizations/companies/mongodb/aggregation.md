# MongoDB Aggregation

Aggregation operations process multiple documents and return computed results.

📌 [MongoDB docs](https://www.mongodb.com/docs/manual/aggregation/)

See also: [Practical MongoDB Aggregations Book](https://www.practical-mongodb-aggregations.com/)

## Aggregation Pipeline

An aggregation pipeline consists of one or more stages that process documents.  
These documents can come from a collection, a view, or a specially designed stage.

🌐 [MongoDB docs](https://www.mongodb.com/docs/manual/core/aggregation-pipeline/)
