# Wolfi

> Wolfi is a Linux OS designed for the container and cloud-native era.
> Chainguard started the Wolfi project to enable building Chainguard Containers, our collection of curated distroless images that meet the requirements of a secure software supply chain.

🌐 [GitHub](https://github.com/wolfi-dev)

## Source

[wolfi-dev/os](https://github.com/wolfi-dev/os) is the main package repository for the Wolfi project.
