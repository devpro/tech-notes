## Frameworks

### SITF

🌐 [wiz-sec-public.github.io](https://wiz-sec-public.github.io/SITF/), [wiz.io/blog](https://www.wiz.io/blog/sitf-sdlc-threat-framework)

The SDLC Infrastructure Threat Framework (STIF) fills the gap when MITRE ATT&CK moves too slowly for supply chain threats:

- Covers 5 areas (Endpoint, VC, CI/CD, Registry, Production) across 3 attack stages (Initial Access, Discovery & Lateral Movement and Post-Compromise).
- Provides specific, actionable techniques (like Git Tag Manipulation) and pairs them with direct protective controls.
