### Recipes

#### Replace Timestamp by another field

```json
filter {
  date {
    match => ["@t", "yyyy-MM-dd'T'HH:mm:ss.SSSSSSS'Z'"]
    target => "@timestamp"
  }
}
```

See [plugins-filters-date](https://www.elastic.co/guide/en/logstash/current/plugins-filters-date.html)
