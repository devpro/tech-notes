# Nutanix Cloud Platform

> Nutanix Cloud Platform (NCP) is a comprehensive hybrid cloud infrastructure solution.
> The Nutanix Cloud Platform combines the core elements of its HCI technology with additional cloud services, automation, and integrations, enabling organizations to build, manage, and optimize their IT infrastructure across on-premises, edge, and public cloud environments

🌐 [nutanix.com/products/cloud-platform](https://www.nutanix.com/products/cloud-platform)

## Bundles

### Foundation

[Nutanix Cloud Infrastructure (NCI)](nci.md) provides:

- Compute: Hypervisors (e.g., AHV, VMware, Hyper-V) for VMs & containers
- Storage: Scale-out HCI (e.g., Files, Objects, Blocks) with built-in DR
- Networking: Virtual networking, security (microsegmentation), & edge support (NCI-Edge)

### Operations

[Nutanix Cloud Manager (NCM)](ncm.md) delivers:

- Automation: Lifecycle management, self-service provisioning
- Monitoring: Unified visibility across multicloud (NCM-Edge for remote sites)
- Optimization: Cost governance, capacity planning

### Unified Experience

Nutanix Central:

- Single pane for NCP/NCI/NCM views
- Manages services like databases & storage across environments

## Extensions

Build on NCP/NCI for specialized workloads:

- [Nutanix Kubernetes Platform (NKP)](nkp.md): Container orchestration & management
- [Nutanix Unified Storage (NUS)](nus.md): File/block/object storage for data-intensive apps
- [Nutanix Database Service (NDB)](ndb.md): Multicloud DB ops (e.g., Oracle, MySQL) with cloning & scaling
- Nutanix Data Lens (NDL): Data governance & compliance
- End-User Computing (EUC): VDI/Desktop-as-a-Service via NCM for EUC
